# 质数
## 分解质因数
一个数最小的因子一定是质数。
```c++
 for (int i = 2; i <= x / i; i ++ )
        if (x % i == 0)
        {
            int s = 0;
            while (x % i == 0) x /= i, s ++ ;
            cout << i << ' ' << s << endl;
        }
    if (x > 1) cout << x << ' ' << 1 << endl;
    cout << endl;
```
## 筛质数
线性筛：
```c++
for(int i=2;i<=n;i++)
    {
        if(!st[i])primes[cnt++]=i;
        for(int j=0;primes[j]<=n/i;j++)
        {
            st[primes[j]*i]=true;
            if(i%primes[j]==0)break;
        }
    }
```

## 最大公约数
辗转相除法：
```c++
int gcd(int a,int b)
{
  return b?gcd(b,a%b):a;
}

```
